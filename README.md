# My Sport - 스포츠 실시간 데이터 대시보드

실시간 스포츠 데이터를 보여주는 반응형 웹 애플리케이션입니다.

## 기능

### 메인 페이지
- 4개의 영역으로 구성 (상하좌우 배치)
- 1, 2, 3번 영역은 동적으로 배치 (야구, 배구, 국제스포츠)
- 4번 영역은 고정
- 시즌 상태에 따라 1번 영역 자동 배치:
  - 국제스포츠 시즌 중: 국제스포츠가 1번 영역
  - 야구 시즌 중: 야구가 1번 영역
  - 배구 시즌 중: 배구가 1번 영역

### 야구 영역 (한화이글스)
- 현재 순위 및 전적/승률
- 시즌 중: 현재 진행 중인 시리즈 결과
- 시즌 종료: 최종 순위 및 마지막 시리즈 결과
- 상세 페이지:
  - 리그 순위 (왼쪽 40%)
  - 투수/타자 기록 탭 (오른쪽 60%)
  - 시즌 중: 현재 시리즈 전적
  - 상대전적 (하단)

### 배구 영역 (현대캐피탈 스카이워커스)
- 시즌 순위
- 전적/승률/세트득실률
- 최근 경기 결과
- 다음 경기 일정
- 상세 페이지:
  - 리그 순위 및 다음 예정 경기 (왼쪽 40%)
  - 시즌 중: 지난 경기 결과 (2주, 오른쪽 60%)
  - 시즌 종료: 공격수 순위

### 데이터 업데이트
- 시즌 중: 오전 10시, 오후 10시 (야구, 배구)
- 시즌 종료: 일주일에 한번
- 새 시즌 시작 1주일 전부터 D-7, D-6 등 표시

## 기술 스택
- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router

## 시작하기

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

### 빌드
```bash
npm run build
```

## 프로젝트 구조

```
src/
├── components/          # 컴포넌트
│   ├── Baseball/        # 야구 영역 컴포넌트
│   ├── Volleyball/      # 배구 영역 컴포넌트
│   ├── InternationalSports/  # 국제스포츠 영역 컴포넌트
│   └── MainLayout/      # 메인 레이아웃
├── pages/               # 페이지 컴포넌트
│   ├── MainPage.tsx
│   ├── BaseballDetail.tsx
│   ├── VolleyballDetail.tsx
│   └── InternationalSportsDetail.tsx
├── types/               # TypeScript 타입 정의
├── utils/               # 유틸리티 함수
│   ├── seasonManager.ts # 시즌 관리 로직
│   └── dataUpdater.ts   # 데이터 업데이트 로직
├── App.tsx
└── main.tsx
```

## 반응형 디자인
- 모바일: 세로 배치 (1열)
- 태블릿/데스크톱: 그리드 배치 (2열)
